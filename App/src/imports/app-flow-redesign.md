## 목표
기존 디자인에서 세 가지를 수정한다.
1. 비즈니스 프로필 페이지 상단 버튼 제거
2. 사이드바 온보딩 설정 메뉴 제거
3. 앱 최초 실행 플로우 신규 화면 추가
   (이메일 연동 → 온보딩 → 템플릿 생성 → 메인 진입)

---

## 1. 비즈니스 프로필 페이지 수정

- 페이지 우측 상단 "템플릿 전체 재생성" 버튼 제거
- 버튼 제거 후 헤더 영역은 제목과 부제목만 남기고 깔끔하게 유지
- 나머지 페이지 구성은 변경 없음

---

## 2. 사이드바 메뉴 수정

- 사이드바에서 "온보딩 설정" 메뉴 항목 제거
- 제거 후 메뉴 순서:
    대시보드
    템플릿 라이브러리
    수신함 자동화
    캘린더
    비즈니스 프로필
    자동화 설정
- 비즈니스 프로필 페이지 최하단에 작은 텍스트 링크 추가
  "초기 온보딩 설정 다시 보기 →" (회색 텍스트, 작게)

---

## 3. 최초 실행 플로우 신규 화면 추가

앱을 처음 설치하고 실행했을 때 진입하는 화면 시퀀스를 디자인한다.
기존 온보딩 디자인 시스템(색상, 폰트, 카드 스타일) 동일하게 적용.
상단에 전체 진행 단계 표시:
  [이메일 연동] → [비즈니스 설정] → [템플릿 생성] → [시작]

### 화면 1: 이메일 계정 연동

상단 스텝바: 1단계 "이메일 연동" 활성화 상태

중앙 카드:
  제목: "업무 이메일 계정을 연결해주세요"
  부제목: "수신 이메일 자동화를 시작하려면 이메일 계정 연동이 필요합니다"

  연동 방식 선택 (3개 카드, 가로 나열):
    [Gmail 아이콘]        [Outlook 아이콘]      [기타 메일]
    Gmail                Microsoft Outlook     IMAP 직접 연결
    Google 계정으로       Microsoft 계정으로    네이버, 다음,
    간편 연결             간편 연결             회사 메일 등

  선택 시 해당 카드 teal 테두리 활성화

  Gmail/Outlook 선택 시 하단:
    "[Gmail 아이콘] Google 계정으로 로그인" 버튼 (primary)

  IMAP 선택 시 하단 폼 노출:
    이메일 주소 입력
    IMAP 서버 주소 입력
    포트 번호 입력
    비밀번호 / 앱 비밀번호 입력
    "[이메일 연결하기]" 버튼 (primary)

  연결 완료 시:
    카드 중앙에 teal 체크 아이콘 + "연결 완료" 텍스트
    연결된 이메일 주소 표시 (예: user@gmail.com)
    "[다음 단계로 →]" 버튼 활성화

  하단 보조 텍스트:
    "이메일 원문은 외부 서버에 저장되지 않습니다"
    (자물쇠 아이콘과 함께, 회색 소형 텍스트)

### 화면 2 ~ 4: 기존 온보딩 3단계

상단 스텝바: 2단계 "비즈니스 설정" 활성화 상태로 변경
기존 온보딩 설정 화면(회사 프로필 → 비즈니스 자료 → 이메일 카테고리) 그대로 유지
스텝바만 기존 3단계에서 전체 4단계 기준으로 교체

### 화면 5: 템플릿 생성 중 로딩

상단 스텝바: 3단계 "템플릿 생성" 활성화 상태
기존에 제안된 템플릿 생성 로딩 화면 디자인 동일하게 적용

### 화면 6: 시작 완료

상단 스텝바: 4단계 "시작" 활성화 상태

중앙 카드:
  상단: teal 체크 원형 아이콘 (크게)
  제목: "모든 설정이 완료되었습니다"
  부제목: "이제 수신 이메일을 자동으로 분류하고 답변 초안을 생성할 준비가 되었습니다"

  완료 항목 요약 (체크리스트):
    ✓ Gmail 연동 완료 (user@gmail.com)
    ✓ 비즈니스 프로필 설정 완료
    ✓ 템플릿 8개 생성 완료

  하단 버튼:
    Primary: "대시보드로 이동 →"

---

## 디자인 유의사항

- 전체 플로우에서 상단 스텝바는 항상 동일한 위치에 고정
- 스텝바 완료 단계는 teal 체크 아이콘, 현재 단계는 네이비 배경,
  미완료 단계는 회색으로 표시
- 이메일 연동 화면의 3가지 연동 카드는
  기존 온보딩의 이메일 어조 선택 카드와 동일한 스타일 적용
- 한국어 텍스트 사용
- 기존 온보딩 및 자동화 설정 화면과 동일한 디자인 시스템 유지
