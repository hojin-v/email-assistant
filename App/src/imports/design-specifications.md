## 목표
전역 검색, 알림 패널, 프로필 드롭다운 메뉴를
실제 동작 기준에 맞게 디자인한다.
검색은 현재 페이지 컨텍스트에 따라 동작이 달라지는 구조로 설계한다.

---

## 1. 전역 검색창

### 기본 상태 (아무 페이지)
  플레이스홀더: "검색..."
  클릭 시: 검색창 확장 + 배경 dimmed 처리

### 페이지별 검색 동작 분기

검색창 클릭 시 현재 페이지에 따라 다른 드롭다운 표시:

  [수신함 페이지]
    드롭다운 헤더: "이메일 검색"
    검색 입력 시 실시간 결과:
      각 결과 행 구성:
        발신자 아바타 (이니셜 원형)
        발신자명 + 회사명 (굵게 / 회색)
        이메일 제목 (키워드 하이라이트, teal)
        수신 날짜 (회색, 소형, 우측)
      결과 행 클릭 시:
        수신함으로 이동 + 해당 이메일 선택 상태로 열람

  [캘린더 페이지]
    드롭다운 헤더: "일정 검색"
    검색 입력 시 실시간 결과:
      각 결과 행 구성:
        캘린더 라인 아이콘 (teal)
        일정 제목 (키워드 하이라이트, teal)
        날짜 + 시간 (회색, 소형)
        출처 배지: [이메일 감지] or [직접 등록]
      결과 행 클릭 시:
        캘린더로 이동 + 해당 날짜로 달력 이동
        + 일정 상세 패널 열람

  [템플릿 라이브러리 페이지]
    드롭다운 헤더: "템플릿 검색"
    검색 입력 시 실시간 결과:
      각 결과 행 구성:
        카테고리 컬러닷 + 카테고리 배지
        템플릿 제목 (키워드 하이라이트, teal)
        템플릿 본문 첫 줄 미리보기 (회색, 소형)
      결과 행 클릭 시:
        템플릿 라이브러리에서 해당 템플릿 카드 하이라이트
        + 템플릿 상세 모달 오픈

### 공통 드롭다운 요소
  검색 결과 없을 때:
    "검색 결과가 없습니다" (회색, 중앙 정렬)
  검색창 외부 클릭 시:
    드롭다운 닫힘, 배경 dimmed 해제

## 2. 알림 패널

### 알림 아이콘
  미확인 알림 있을 때: teal 원형 배지 (숫자 표시)
  없을 때: 배지 숨김

### 알림 아이콘 클릭 시
  우측에서 슬라이드로 열리는 패널
  패널 너비: 360px
  배경: 흰색
  좌측 나머지 화면: 반투명 dimmed 처리

### 패널 헤더
  제목: "알림"
  우측: "모두 읽음 처리" (teal 텍스트 링크, 소형)

### 알림 탭
  [전체] [미확인]
  활성 탭: teal 하단 밑줄

### 알림 항목 종류 (각각 디자인)

  새 이메일 + 템플릿 매칭 완료:
    아이콘: 이메일 라인 아이콘 (teal 배경 원형)
    제목: "새 이메일이 도착했습니다"
    내용: "박민수 · 엔터프라이즈 플랜 가격 문의"
           "가격문의 템플릿 매칭 완료"
    시간: "5분 전" (회색, 소형)
    우측: [확인하기 →] teal 텍스트 링크
    클릭 시 수신함 해당 이메일로 이동

  검토 대기 초안 누적:
    아이콘: 문서 라인 아이콘 (amber 배경 원형)
    제목: "검토 대기 중인 초안이 있습니다"
    내용: "3개의 초안이 검토를 기다리고 있습니다"
    시간: "10분 전"
    우측: [바로 확인 →] amber 텍스트 링크
    클릭 시 수신함 대기 중 탭으로 이동

  이메일 계정 연결 오류:
    아이콘: 경고 라인 아이콘 (red 배경 원형)
    제목: "이메일 계정 연결이 끊겼습니다"
    내용: "user@gmail.com 재연결이 필요합니다"
    시간: "1시간 전"
    우측: [재연결 →] red 텍스트 링크
    클릭 시 계정 설정으로 이동

  미분류 이메일 발생:
    아이콘: 물음표 라인 아이콘 (회색 배경 원형)
    제목: "분류되지 않은 이메일이 있습니다"
    내용: "2개의 이메일이 어떤 카테고리에도
           매칭되지 않았습니다"
    시간: "30분 전"
    우측: [카테고리 설정 →] 회색 텍스트 링크

  캘린더 등록 대기:
    아이콘: 캘린더 라인 아이콘 (teal 배경 원형)
    제목: "등록 대기 중인 일정이 있습니다"
    내용: "테크솔루션 엔터프라이즈 플랜 상담
           3월 4일 14:00"
    시간: "20분 전"
    우측: [일정 확인 →] teal 텍스트 링크

  자동 발송 일간 요약:
    아이콘: 번개 라인 아이콘 (green 배경 원형)
    제목: "오늘 자동 발송 요약"
    내용: "총 39건이 자동 발송되었습니다"
    시간: "오늘 오후 6:00"
    우측: [상세 보기 →] green 텍스트 링크

### 미확인 알림 스타일
  행 배경: 연한 teal (#F0FAFA)
  읽은 알림: 흰색 배경

### 패널 하단
  "알림 설정 →" (회색 텍스트 링크, 중앙 정렬)
  클릭 시 프로필 메뉴 > 알림 설정으로 이동

## 3. 프로필 드롭다운 메뉴

### 프로필 아이콘 클릭 시
  아이콘 바로 아래 드롭다운 메뉴 오픈
  너비: 240px
  배경: 흰색
  그림자: subtle drop shadow
  라운드: 12px

### 드롭다운 구성

  상단 사용자 정보 영역:
    배경: 연한 회색 (#F8FAFC)
    프로필 아바타 (이니셜 원형, 네이비, 좌측)
    우측:
      이름: "김호진" (굵게, 네이비)
      역할: "관리자" (회색, 소형)
      이메일: "user@gmail.com" (회색, 소형)

  구분선

  계정 설정 섹션 (섹션 라벨: "계정", 회색 소형):
    🔗 이메일 계정 연동 관리
       (자동화 설정에서 이동해온 항목)
    🔑 로그인 정보 변경
    🔔 알림 설정

  구분선

  화면 설정 섹션 (섹션 라벨: "화면", 회색 소형):
    ⊞  대시보드 위젯 설정
       (카드/섹션 순서 커스텀)
    🌐 언어 설정
    🌙 테마 설정
       우측에 [라이트 / 다크] 토글 인라인 표시

  구분선

  🚪 로그아웃 (red 텍스트)

### 각 메뉴 항목 스타일
  높이: 40px
  좌측 아이콘 (라인 아이콘, 회색) + 텍스트 (네이비)
  호버 시: 연한 회색 배경
  라운드: 8px

---

## 4. 자동화 설정 페이지 수정

  이메일 계정 연동 섹션을 자동화 설정에서 제거
  (프로필 메뉴 > 계정 설정으로 이동했으므로)
  해당 섹션 자리에 안내 텍스트 추가:
    "이메일 계정 연동은 프로필 > 계정 설정에서 관리합니다 →"
    (회색 소형 텍스트 링크)

---

## 디자인 유의사항

- 검색 드롭다운, 알림 패널, 프로필 드롭다운이
  동시에 열리지 않도록 설계
  (하나가 열리면 나머지는 닫힘)
- 알림 패널과 프로필 드롭다운은
  외부 클릭 시 닫힘 처리
- 모든 아이콘은 이모티콘 대신 라인 아이콘 사용
- 한국어 텍스트 사용
- 기존 네이비 사이드바, teal 액센트 디자인 시스템 유지
