## 목표
온보딩 마지막 단계(이메일 카테고리) 이후 템플릿 생성 로딩 화면과 
완료 화면을 추가하고, 온보딩 완료 후 동일한 내용을 
언제든 수정할 수 있는 "비즈니스 프로필" 설정 페이지를 새로 디자인한다.

---

## 1. 온보딩 카테고리 단계 수정

기존 이메일 카테고리 설정 화면 하단 버튼을 아래와 같이 수정한다.

- 기존 버튼 텍스트 "설정 완료" → "템플릿 생성 시작"으로 변경
- 버튼 하단에 보조 안내 문구 추가
  "설정한 카테고리를 기반으로 AI가 맞춤 템플릿을 자동 생성합니다"

---

## 2. 템플릿 생성 중 로딩 화면 (신규)

"템플릿 생성 시작" 클릭 후 전환되는 풀페이지 화면.
기존 온보딩 디자인 시스템(색상, 폰트, 카드 스타일) 동일하게 적용.

### 레이아웃
- 중앙 정렬 단일 카드
- 상단: 애니메이션 스피너 또는 프로그레스 바
- 제목: "맞춤 템플릿을 생성하고 있습니다"
- 부제목: "입력하신 비즈니스 정보를 분석하여 카테고리별 템플릿을 만들고 있습니다"

### 단계별 진행 표시 (체크리스트 형태)
  ✓ 회사 프로필 분석 완료       (완료 상태 - teal 체크 아이콘)
  ✓ 비즈니스 자료 학습 완료     (완료 상태 - teal 체크 아이콘)
  ⟳ 카테고리별 템플릿 생성 중  (진행 상태 - 스피너 아이콘)
    가격문의 ████░░ 3/5
  ○ 키워드 및 분류 규칙 설정    (대기 상태 - 회색 원)

### 하단
- 예상 소요 시간 안내: "약 30초~1분 소요됩니다"
- 취소 링크 (ghost 텍스트): "취소하고 나중에 설정하기"

---

## 3. 템플릿 생성 완료 화면 (신규)

로딩 완료 후 전환되는 화면.

### 레이아웃
- 중앙 정렬 단일 카드
- 상단: 완료 아이콘 (teal 계열 체크 원형 아이콘, 크게)
- 제목: "8개의 템플릿이 생성되었습니다"
- 부제목: "카테고리별 템플릿을 확인하고 필요시 수정해보세요"

### 생성 결과 요약 (카테고리별 칩 나열)
  가격문의 5개  /  불만접수 4개  /  미팅요청 3개  /  기술지원 4개 ...
  (각 카테고리 칩은 온보딩 카테고리 태그와 동일한 컬러 닷 스타일 적용)

### 안내 문구 박스 (연한 teal 배경)
  💡 템플릿은 언제든지 수정하거나 새로 추가할 수 있습니다.
     비즈니스 자료가 변경되면 템플릿을 재생성할 수 있습니다.

### 하단 버튼
- Primary: "템플릿 확인하기 →" (템플릿 라이브러리로 이동)
- Secondary: "이메일 계정 연결하기" (다음 설정 단계로 이동)

---

## 4. 비즈니스 프로필 설정 페이지 (신규)

사이드바 메뉴에 "비즈니스 프로필" 항목을 추가하고,
온보딩에서 입력한 모든 정보를 언제든 수정할 수 있는 설정 페이지를 디자인한다.
기존 자동화 설정 페이지와 동일한 레이아웃 구조(페이지 제목, 섹션 카드) 적용.

### 페이지 헤더
- 제목: "비즈니스 프로필"
- 부제목: "AI 템플릿 생성에 사용되는 비즈니스 정보를 관리합니다"
- 우측 상단: "템플릿 전체 재생성" 버튼 (outline 스타일)

### 섹션 1: 회사 프로필 카드
현재 설정값을 보여주고 인라인 수정 가능한 폼 형태.

  업종 / 비즈니스 유형  [드롭다운 - 현재값 표시]
  이메일 어조           [격식체 / 중립 / 친근한 선택 칩]
  제품/서비스 설명      [텍스트에어리어 - 현재 입력값 표시]

  우측 하단: "저장" 버튼

### 섹션 2: 비즈니스 자료 카드
업로드된 파일과 직접 입력한 FAQ를 목록으로 표시.

  [업로드된 파일]
  각 파일은 행(row) 형태로 표시:
    📄 아이콘  파일명.pdf    2025.01.15 업로드    [삭제 아이콘]
    📄 아이콘  매뉴얼.docx   2025.03.01 업로드    [삭제 아이콘]
  하단: "+ 파일 추가" 버튼 (dashed border 스타일)

  [FAQ / 매뉴얼 직접 입력]
  각 항목은 행(row) 형태로 표시:
    Q: 환불 정책은? / A: 14일 이내...    [수정 아이콘] [삭제 아이콘]
  하단: "+ 항목 추가" 버튼

### 섹션 3: 자료 변경 감지 알림 배너
비즈니스 자료가 수정된 경우에만 섹션 2 하단에 표시되는 경고 배너.

  ⚠️ 비즈니스 자료가 변경되었습니다.
     영향받는 템플릿 8개가 있습니다.
  [템플릿 일괄 재생성]  [개별 선택 후 재생성]  [나중에]

  배너 배경색: 연한 amber/yellow 계열
  버튼: Primary / Secondary / Ghost 순서

---

## 디자인 유의사항

- 기존 온보딩 및 자동화 설정 화면과 동일한 디자인 시스템 유지
  (네이비, teal 액센트, 라운드 카드, 회색 배경)
- 한국어 텍스트 사용
- 로딩/완료 화면은 온보딩 3단계 스텝바 제거하고 독립 화면으로 처리
- 비즈니스 프로필 페이지는 기존 사이드바 네비게이션에 자연스럽게 추가
