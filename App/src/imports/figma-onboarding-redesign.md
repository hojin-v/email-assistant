## Figma Make 프롬프트

```
## 목표
최초 실행 첫 화면(이메일 연동 단계)을 상용 SaaS 서비스 수준의 
세련된 온보딩 화면으로 전면 재디자인한다.
이후 단계(비즈니스 설정, 템플릿 생성, 시작)도 동일한 새 디자인 
시스템을 적용한다.

---

## 전체 레이아웃 구조 변경

### 기존 구조 (변경 전)
- 상단 스텝바 + 중앙 카드만 있는 단순 구조
- 여백이 많고 카드가 상단에 쏠려 있음

### 새 구조 (변경 후)
좌우 2단 분할 레이아웃으로 전환한다.

```
┌─────────────────┬──────────────────────────┐
│                 │                          │
│   좌측 패널     │      우측 패널            │
│   (40%)         │      (60%)               │
│                 │                          │
│   브랜드 영역   │   이메일 연동 카드         │
│                 │                          │
└─────────────────┴──────────────────────────┘
```

---

## 좌측 패널 디자인

배경: 진한 네이비(#1E2A3A) 단색 또는 
네이비에서 약간 밝은 네이비로의 subtle한 그라데이션
전체 높이 100vh 꽉 채움

구성 요소 (수직 중앙 정렬):

  상단:
    서비스 로고 + 서비스명 "업무용 이메일 자동화"
    (로고는 teal 계열 아이콘, 텍스트는 흰색)

  중간 (메인 카피):
    대형 텍스트 (흰색, 굵게):
    "이메일 업무를
     AI가 대신합니다"

    소형 설명 텍스트 (연한 회색):
    "분류부터 답변 초안까지,
     반복되는 이메일 업무를 자동화하세요"

  하단 (신뢰 요소):
    작은 아이콘 + 텍스트 3줄 (연한 회색, 소형):
    🔒  이메일 원문은 외부 서버에 저장되지 않습니다
    ⚡  평균 설정 시간 3분
    ✓   언제든 설정 변경 가능

---

## 우측 패널 디자인

배경: 밝은 회색(#F8FAFC)
전체 높이 100vh, 수직 수평 중앙 정렬

상단 스텝바:
  좌측 상단 고정 (우측 패널 내부 상단)
  1 이메일 연동 (활성) → 2 비즈니스 설정 → 3 템플릿 생성 → 4 시작
  활성 단계: teal 배경 흰색 텍스트
  완료 단계: teal 테두리 + 체크 아이콘
  미완료 단계: 회색 테두리 회색 텍스트

중앙 콘텐츠 카드:
  흰색 배경, 라운드 16px, subtle 그림자
  패딩 넉넉하게 (40px 이상)
  카드 너비 우측 패널의 75% 수준

  카드 내부:
    제목: "업무 이메일 계정을 연결해주세요" (네이비, 굵게)
    부제목: "수신 이메일 자동화를 시작하려면 이메일 계정 연동이 필요합니다"
            (회색, 소형)

    연동 방식 선택 카드 3개 (가로 나열):
      각 카드: 흰색 배경, 회색 테두리, 라운드 12px
      호버 시: teal 테두리로 변경
      선택 시: teal 테두리 + 연한 teal 배경

      [Gmail]
        실제 Gmail 로고 컬러 아이콘 (빨강)
        "Gmail"
        "Google 계정으로 간편 연결"

      [Microsoft Outlook]
        실제 Outlook 로고 컬러 아이콘 (파랑)
        "Microsoft Outlook"
        "Microsoft 계정으로 간편 연결"

      [기타 메일]
        일반 메일 아이콘 (회색)
        "기타 메일"
        "네이버, 다음, 회사 메일 등"

    카드 하단 여백 후:
      선택에 따라 연결 버튼 또는 IMAP 폼 노출 (기존 명세 동일)

## 비즈니스 설정 단계 (2단계) 동일 레이아웃 적용

좌측 패널:
  브랜드 영역 유지하되 중간 텍스트 변경:
    대형 텍스트: "비즈니스 정보를
                 입력해주세요"
    소형 텍스트: "입력한 정보를 바탕으로
                 AI가 맞춤 템플릿을 생성합니다"
  하단 신뢰 요소 동일 유지

우측 패널:
  스텝바 2단계 활성 상태
  기존 회사 프로필 설정 폼 카드 그대로 유지
  이전/다음 버튼은 카드 하단 외부에 배치
  (이전: ghost 텍스트 버튼 좌측 / 다음: primary 버튼 우측)

---

## 디자인 유의사항

- 좌측 패널은 모든 단계에서 동일하게 유지
  (텍스트 내용만 단계별로 변경)
- 전체적으로 Linear, Notion, Clerk 같은
  모던 B2B SaaS 온보딩 느낌을 참고
- 폰트 크기 위계를 명확히:
  좌측 대형 카피 32~36px / 카드 제목 20px / 본문 14px
- 불필요한 장식 요소 없이 여백과 타이포그래피로 고급스러움 표현
- 한국어 텍스트 사용
- 기존 teal 액센트 컬러 (#2DD4BF) 유지
```
