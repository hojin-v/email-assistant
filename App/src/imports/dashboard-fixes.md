## 목표
대시보드 3가지 문제를 수정한다.
1. 검토 대기 배너와 카드 중복 제거
2. 다가오는 일정 섹션 승인 상태 반영
3. 디자인 일관성 개선 (이모티콘, 배지 스타일)

---

## 1. 검토 대기 중복 표시 제거

### 문제
상단 액션 배너 "검토 대기 중인 초안 2개가 있습니다"와
지표 카드 2번 "검토 대기 중인 초안 3 / 즉시 확인 필요"가
동일한 정보를 중복으로 표시하고 있음

### 변경 방향
상단 액션 배너를 제거하고
지표 카드 2번이 단독으로 대기 상태를 전달하도록 한다

### 지표 카드 2번 강조 강화
배너가 없어지므로 카드가 더 눈에 띄어야 함

  대기 중인 초안이 있을 때:
    카드 테두리: amber 2px solid
    카드 배경: 연한 amber (#FFFBEB)
    아이콘: amber 유지
    숫자: amber로 변경 (기존 검정에서)
    "즉시 확인 필요" 텍스트 유지
    카드 전체 클릭 시 수신함 대기 중 탭으로 이동

  대기 중인 초안이 0개일 때:
    카드 테두리/배경 기본 흰색으로 복귀
    숫자: 0 (검정)
    하단 보조 텍스트: "모두 처리 완료" (회색)

---

## 2. 다가오는 일정 섹션 수정

### 문제
사용자가 캘린더 등록을 승인하지 않은 일정도
감지되는 즉시 대시보드에 표시되어 정보 혼선 발생

### 변경 방향
캘린더 등록 승인 여부에 따라 일정을 구분하여 표시한다

### 일정 항목 상태 2종류

  승인 완료된 일정 (캘린더에 등록됨):
    기존 디자인 유지
    좌측 컬러 보더 그대로
    이메일 감지 배지 제거 (이미 등록 완료이므로 불필요)

  감지됐으나 미승인 일정 (등록 대기 중):
    좌측 컬러 보더를 점선으로 변경
    일정 제목 옆에 [등록 대기] 배지 추가
      배경: 연한 amber, 텍스트: amber, 소형
    우측에 "캘린더에 추가" 버튼 표시
      teal outline 소형 버튼
      클릭 시 캘린더 연동 확인 모달 오픈
    "이메일 감지" 배지는 미승인 항목에만 표시

### 섹션 헤더 수정
  기존: 📅 다가오는 일정  [이메일 연동]  캘린더 보기 ↗
  변경:
    아이콘: 이모티콘 제거 → 라인 아이콘으로 교체
            (캘린더 라인 아이콘, teal)
    [이메일 연동] 배지 → 하단 디자인 시스템에 맞게 수정
            (아래 배지 통일 기준 참고)
    우측: "캘린더 보기 →" (teal 텍스트 링크, 화살표 이모티콘 제거)

## 3. 디자인 일관성 수정

### 이모티콘 전체 제거
대시보드 내 모든 이모티콘을 라인 아이콘으로 교체한다

  📅 다가오는 일정  →  캘린더 라인 아이콘 (teal)
  그 외 이모티콘 사용 부분 동일하게 교체

### [이메일 연동] 배지 스타일 통일
기존 보라색 pill 배지를 다른 상태 배지들과 동일한 스타일로 교체

  변경 전: 보라색 배경 "이메일 연동" 텍스트 배지
  변경 후:
    배경: 연한 회색 (#F3F4F6)
    텍스트: 중간 회색
    테두리: 회색 1px
    아이콘: 이메일 라인 아이콘 (소형, 회색) + "이메일 연동" 텍스트
    라운드: 6px

### 최근 수신 이메일 섹션 헤더
  기존: 최근 수신 이메일  전체 보기 ↗
  변경:
    이모티콘 없음 유지 (현재 없으므로 그대로)
    "전체 보기 →" 화살표 이모티콘 제거, teal 텍스트 링크만 유지

---

## 디자인 유의사항

- 대시보드 내 모든 섹션 헤더 아이콘을
  동일한 스타일의 라인 아이콘으로 통일
- 배지 스타일 기준:
  상태 배지 (검토 대기/처리 완료 등): 컬러 배경
  정보성 배지 (이메일 연동 등): 회색 배경
- 한국어 텍스트 사용
- 기존 네이비 사이드바, teal 액센트 디자인 시스템 유지
