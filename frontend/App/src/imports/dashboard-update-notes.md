## 목표
대시보드 화면을 실용적인 정보 중심으로 개선한다.
불필요한 추상 지표를 제거하고, 사용자가 즉시 액션할 수 있는
정보 위계로 재구성한다. 기존 디자인 시스템 유지.

---

## 1. 상단 액션 배너 (신규 추가)

위치: 대시보드 최상단, 지표 카드보다 위
조건: 검토 대기 중인 초안이 있을 때만 표시
없을 때는 배너 영역 자체가 사라지고 지표 카드가 최상단으로 올라옴

디자인:
  배경: 연한 teal (#E6FAF8) 
  좌측: ✉ 아이콘 (teal) + 텍스트
    "검토 대기 중인 초안 3개가 있습니다"
    (굵게, 네이비)
  우측: "바로 확인 →" 버튼 (teal, small)
  라운드 12px, 패딩 16px

## 2. 상단 지표 카드 4개 수정

### 기존 카드 구성 (변경 전)
  오늘 처리된 이메일 / 자동 응답률 / 평균 응답 시간 / 고객 만족도

### 새 카드 구성 (변경 후)
카드 순서와 내용을 아래와 같이 교체한다.

  카드 1: 오늘 처리된 이메일
    아이콘: 이메일 아이콘 (teal)
    숫자: 47
    라벨: "오늘 처리된 이메일"
    하단 보조: "어제 대비 +5" (회색 소형 텍스트, 증감 기준 명시)

  카드 2: 검토 대기 중인 초안
    아이콘: 문서 아이콘 (amber/주황)
    숫자: 3
    라벨: "검토 대기 중인 초안"
    하단 보조: "즉시 확인 필요" (amber 소형 텍스트)
    카드 전체에 amber 계열 subtle 테두리 추가
    (대기 중인 초안이 0개면 테두리 제거, 보조 텍스트 "없음"으로 변경)

  카드 3: 템플릿 매칭률
    아이콘: 타겟/체크 아이콘 (teal)
    숫자: 96%
    라벨: "템플릿 매칭률"
    하단 보조: "전주 대비 +2%" (회색 소형 텍스트, 기준 명시)

  카드 4: 이메일 계정 상태
    아이콘: 연결 아이콘 (green)
    숫자 대신: "정상 연결" 텍스트 (green, 굵게)
    라벨: "이메일 계정 상태"
    하단 보조: "user@gmail.com" (회색 소형 텍스트)
    오류 상태일 때: 아이콘/텍스트 red로 변경,
    "연결 오류 - 재연결 필요" + [재연결] 링크 표시

## 4. 최근 수신 이메일 섹션 수정

### 변경할 항목
- 각 이메일 행 우측의 "매칭률 96%" 제거
- 대신 처리 상태 배지로 교체

### 처리 상태 배지 종류
  [초안 대기 중]  - amber 배경, 진한 amber 텍스트
  [자동 발송됨]   - green 배경, 진한 green 텍스트
  [처리 완료]     - 회색 배경, 회색 텍스트

### 각 이메일 행 구성 (좌→우)
  좌측:
    발신자 아바타 (이니셜, 원형)
    발신자명 · 회사명 (굵게 / 회색)
    이메일 제목 미리보기 (회색, 소형)
    수신 시간 (연한 회색, 소형)

  우측:
    카테고리 배지 (기존 스타일 유지)
    처리 상태 배지 (위 종류 참고)
    "검토하기 →" 링크 텍스트
    (초안 대기 중 상태일 때만 표시, teal 텍스트)

## 5. 이번 주 요약 섹션 (신규 추가)

위치: 최근 수신 이메일 섹션 하단

제목: "이번 주 요약"
우측: 기간 표시 "3월 3일 - 3월 9일" (회색 소형)

내용: 가로 2단 구성

  좌측 (60%): 카테고리별 수신 비율
    도넛 차트
    범례: 각 카테고리명 + 건수 + 비율
