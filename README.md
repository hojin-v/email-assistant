# EmailAssist Frontend

이 디렉터리는 EmailAssist의 프론트엔드 애플리케이션만 모아둔 영역입니다.  
사용자용 서비스 화면과 관리자용 운영 화면을 분리해 관리하며, 두 앱 모두 이메일 업무 자동화를 중심으로 구성됩니다.

## 구성

### `App`

실사용자를 위한 메인 웹 애플리케이션입니다.

- 수신 메일 확인 및 답변 초안 검토
- 템플릿 라이브러리 조회, 생성, 수정
- 일정 감지 및 캘린더 등록
- 자동화 규칙 설정
- 비즈니스 프로필, FAQ, 자료 기반 응답 품질 개선

현재 `App`은 React, Vite, TypeScript 기반으로 작성되어 있습니다.

### `Admin-Web`

운영자와 관리자용 관리 콘솔입니다.

- 운영 대시보드
- 회원 및 부서 권한 구조 확인
- 템플릿 운영 및 승인 대기 확인
- 사용자 문의 대응 기록 관리

현재 `Admin-Web`은 React, Vite 기반 JSX 구조로 유지하고 있습니다.
